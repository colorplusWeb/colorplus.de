Datenübertragung

Daten übertragen mit Cyberduck

Laden Sie sich zunächst das Programm "Cyberduck" aus unseren Download-Bereich herunter. Beim ersten Programmstart sollten Sie folgendes Fenster sehen:

Klicken Sie hier auf "Neue Verbindung"

Neue Verbindung herstellen

Im nächsten Fenster geben Sie bitte folgende Werte ein:
Server: ftp.color-druck-ftp.net
Port: 21
Benutzername: color-druck-ftp.net
Passwort: erhalten Sie telefonisch auf Anfrage

Zugangsdaten

Obwohl der Server das Protokoll FTP-SSH unterstützt sollten Sie dieses nicht wählen. Daher wählen Sie "Fortfahren". Durch aktivieren der Feldes "Nicht mehr zeigen", wird diese Warnmeldung dauerhaft ausgeschaltet.


Im folgenden Fenster sollten Sie 3 Ordner sehen.

Ziehen Sie ihre Dateien per "Drag´n Drop" in den "Kundenupload" Ordner. Die Daten werden nach erfolgreichem Upload automatisch durch unser System heruntergeladen und auf dem Server gelöscht.
